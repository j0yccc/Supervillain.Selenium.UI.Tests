# Supervillain.Selenium.UI.Tests
UI tests for Supervillain project

Requirement:
1. Specflow
2. Selenium 
3. Chrome driver
4. Nunit

Steps:
1. Clone the repo.
2. Open the solution in VS.
3. Build the project. Tests will appear in Test explorer.
4. Run the the tests in Test Explorer.
5. Test results pass/fail will be displayed.
